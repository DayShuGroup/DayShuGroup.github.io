Git

GIT常用命令

  1. git clone 远程仓库地址 从远程仓库克隆到本地目录
  2. git subtree push --prefix=_book origin gh-pages 当前仓库下面的_book目录内容 推送到分支gh-pages(没有则新建)
  3. 分支常用命令
    • git branch 新分支名 创建新分支
    • git checkout 某个分支 切换到某个分支
    • git branch 查看有那些分支及当前分支
